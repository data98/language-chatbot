# Language Coach

Minimal text-only AI language practice app. 
You can check out the live demo [here](https://language-chatbot-project.vercel.app/).

## Stack
- **Next.js 15+** (App Router)
- **Tailwind CSS v4**
- **OpenAI API** (`gpt-4o-mini`)

## Quick Start

1. **Install**
   ```bash
   npm install
   ```

2. **Setup Env**
   Create `.env.local`:
   ```bash
   OPENAI_API_KEY=sk-...
   ```

3. **Run**
   ```bash
   npm run dev
   ```
